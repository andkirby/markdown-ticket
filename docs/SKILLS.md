# Skills Reference

Quick reference for recommended skills in this project.

## Frontend

| Skill | Purpose | When to Use |
|-------|---------|-------------|
| `frontend-react-component` | Design/redesign React components | Creating or refactoring React components |
| `premium-ux-designer` | UI layout, styling, polish | Elevating visual quality, simplifying flows |

## Backend

| Skill | Purpose | When to Use |
|-------|---------|-------------|
| `architecture-patterns` | Backend architecture design | Clean Architecture, Hexagonal, DDD |

Load this skill when mdt:architecture used.

## Testing

| Skill | Purpose | When to Use |
|-------|---------|-------------|
| `playwright-skill` | E2E testing, Playwright patterns | Writing E2E tests, browser automation |
| `webapp-testing` | Manual browser testing/debugging | Live browser interaction, screenshots |

## Code Quality

| Skill | Purpose | When to Use |
|-------|---------|-------------|
| `unified-code-reviewer` | Review code before commit | PRs, feature completion |
| `ts-validate-fix` | Fix TypeScript errors | Type errors, compilation issues |
